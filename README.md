一、项目简介
Unity 2D/3D 休闲烹饪模拟小游戏，使用 C# + Input System + ScriptableObject 开发。玩家限时切菜、煎菜、装盘、送订单，完成菜谱得分。
二、核心玩法流程
开局倒计时，限时内刷出随机订单
从食材柜拿原料，切菜台切割、灶台煎制（超时会烧焦）
盘子装好对应食材，送至出餐台交付订单
多余食材丢垃圾桶销毁
三、核心技术架构
数据驱动（ScriptableObject）
食材、所有菜谱、音效全部做成 SO 配置，不用改代码就能调整关卡、食材、配方。
继承基类复用
KitchenObjectHolder：玩家、所有柜台共用，统一处理拾取、转移、销毁物品
BaseCounter：所有操作台父类，统一交互接口
事件解耦
切菜、上菜、捡物品等动作触发事件，音效、UI、动画只监听事件，逻辑互不干扰。
单例全局管理器
GameManager（控制游戏阶段）、OrderManager（订单生成校验）、SoundManager（全局音效）。
四、主要模块
玩家模块
Input System 控制移动、交互按键；射线检测面前操作台，同步行走动画。
操作台模块
食材柜、切菜台、煎灶台、装盘台、垃圾桶、出餐台、盘子自动生成台，各自实现专属烹饪逻辑。
UI 模块
倒计时、实时订单列表、烹饪进度条、盘子食材图标展示，动态生成界面元素。
音效 & 视觉
SO 统一管理所有音效；操作台配套动画、火焰粒子、3D 盘子模型展示食材。
五、项目优势（简历可用）
采用 SO 数据驱动，策划可快速配置内容
继承 + 事件架构，代码低耦合、易拓展新灶台 / 食材
完整游戏流程：输入、交互、烹饪、订单、UI、音效闭环
